## Build & Run

### Desktop (Python)
1. Install dependencies:
```bash
pip install -r requirements.txt
```
2. Run the application:
```bash
python main.py
```

### Web (Vercel)
1. Push to GitHub
2. Connect to Vercel
3. Deploy automatically

### Windows EXE
1. Run BUILD_WINDOWS_EXE.bat
